## Data Types
- String (str): "Hello"
- Integer (int): 123
- Float (float): 3.14
- Boolean (bool): True / False

## Type Errors
- TypeError happens when you use the wrong data type.
  Example: len(12345) causes an error because len() expects a string/sequence, not an integer.

## Type Checking
- Use type() to check the datatype:
  Example:
    print(type("abc"))  # <class 'str'>

Practice type checks:
- print(type("hello"))
- print(type(123))
- print(type(3.14))
- print(type(True))

## Type Conversion
Convert between types using:
- str()
- int()
- float()

Fixing string + int issue example:
  print("Number of letters in your name: " + str(len(input("Enter your name: "))))

## Mathematical Operations
Operators:
- + addition
- - subtraction
- * multiplication
- / division (returns float)
- // floor division
- ** exponent

## PEMDAS Order
1. Parentheses ()
2. Exponents **
3. Multiplication/Division * / //
4. Addition/Subtraction + -

Example output:
  print(3 * 3 + 3 / 3 - 3)  -> 7.0

Make it output 3:
  print(3 * (3 + 3) / 3 - 3)  -> 3.0

## Number Manipulation
Flooring (remove decimals):
- int(3.738492) -> 3

Rounding:
- round(3.738492) -> 4
- round(3.14159) -> 3
- round(3.14159, 2) -> 3.14

## Assignment Operators
- += add and assign
- -= subtract and assign
- *= multiply and assign
- /= divide and assign

Example:
  score = 0
  score += 1

## f-Strings
Insert values directly inside strings:
  age = 12
  print(f"I am {age} years old")

## Project: Tip Calculator
Goal: Calculate per-person bill with tip and format to 2 decimals.

Example:
- Bill: 150.00
- People: 5
- Tip: 12%

Formula:
- (bill / people) * (1 + tip/100)

Formatting to 2 decimals (recommended):
  amount = (bill / people) * (1 + tip / 100)
  print(f"{amount:.2f}")
