## Randomisation
- Import the random module:
  import random
- Generate a random integer in a range:
  random.randint(0, 2)
- Use random numbers to pick an item from a list by index.

## Lists (Basics)
- Creating a list:
  fruits = ["Cherry", "Apple", "Pear"]
- Accessing items by index (0-based):
  fruits[0] -> "Cherry"
- Negative indexing (from the end):
  fruits[-1] -> last item
- Modifying an item:
  fruits[0] = "Orange"
- Adding an item to the end:
  fruits.append("Mango")

## IndexError (Out of Range)
- If you access an index that does not exist, Python throws IndexError.
  Example:
  states_of_america[50]  # IndexError (if list ends at 49)

## Nested Lists
- A list can contain other lists.
  Example:
  dirty_dozen = [fruits, vegetables]
- Access pattern:
  dirty_dozen[0] -> fruits list
  dirty_dozen[1] -> vegetables list
